# Foundry Padel

Portland's first indoor padel club — 4 courts, café, and social spaces in Portland.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
